# Recepção do equipamento

O módulo de recepção registra cliente, equipamento, defeito relatado, acessórios, checklist físico, fotos e nome para assinatura. A ficha é armazenada localmente no aparelho nesta primeira versão e pode ser impressa imediatamente.

Próxima evolução planejada: vincular a recepção ao Firestore, criar automaticamente cliente/equipamento/OS e enviar fotos ao armazenamento privado.
